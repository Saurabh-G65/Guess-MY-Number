# Guess-MY-Number
A simple guessing game..🤷‍♂️👀✨🏆🎯 

How to play...? 
1. Input a randome number in the input field.
2. Click on " Check! " button.
3. Check the left side Message to get hint of your progress.
4. Follow-up the Score on left side it starts with the Maximum score (20) you can get by winning for every wrong guess it will reduce by one(-1). 
5. Try to keep a higest score 🤞. 
6. Reset your score by clicking on top left button " Again ".
